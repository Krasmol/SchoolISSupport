# Сопровождение информационной системы образовательной организации

## Описание проекта
Приложение для учета заявок в техподдержку, оборудования, программного обеспечения и сотрудников образовательной организации.

## Версии

### AdminVersion
- Полный доступ ко всем таблицам
- CRUD операции (добавление, редактирование, удаление)
- Вход: 'admin' / 'admin123'

### UserVersion
- Только просмотр справочников
- Создание заявок в техподдержку
- Просмотр своих заявок
- Вход: 'user1' / 'user123', 'user2' / 'user123'

## Технологии 
- C# Windows Forms (.NET Framework)
- Microsoft Access (база данных)
- ADO.NET (OleDb)

## Установка 
1. Установите Microsoft Access Database Engine (если ошибка с провайдером)
2. Откройте '.sln' файл в Visual Studio
3. Соберите проект (Ctrl+Shift+B)
4. Запустите (F5)

## База данных
Файл "SchoolIS.accdb' должен находиться в папке 'bin/debug/'
